STRML.net
=========

[View Site](http://hellowor1d.github.io/strml-page/)

Building
--------

```bash
git clone git@github.com:Hellowor1d/strml-page.git
cd strml-page
npm install
npm run dev
# Open localhost:4003/index-dev.html in your browser
```

Building for Production
--------

```bash
npm run build
```
